# Twitter Clone – User Stories

This document outlines the functional user stories for the Twitter (X) clone application.

---

## Authentication & Profile
- As a user, I want to sign up using my email so that I can create an account.
- As a user, I want to log in and log out securely.
- As a user, I want to update my profile (bio and profile picture).
- As a user, I want to view other users’ profiles.

---

## Tweeting & Engagement
- As a user, I want to create tweets so that I can share my thoughts.
- As a user, I want to delete my own tweets.
- As a user, I want to like a tweet.
- As a user, I want to reply to a tweet.
- As a user, I want to retweet a tweet.

---

## Feed & Discovery
- As a user, I want to see a timeline of tweets from people I follow.
- As a user, I want to see my own tweets on my profile.
- As a user, I want to search for users.
- As a user, I want to search for tweets using keywords or hashtags.

---

## Social Connections
- As a user, I want to follow other users.
- As a user, I want to unfollow users.
- As a user, I want to see my followers and following lists.

---

## Notifications
- As a user, I want to receive notifications when someone likes my tweet.
- As a user, I want to receive notifications when someone follows me.
- As a user, I want to receive notifications when someone replies to my tweets.

---

## Admin / Moderation
- As an admin, I want to view all users.
- As an admin, I want to remove inappropriate tweets.
- As an admin, I want to suspend or ban users.

---
