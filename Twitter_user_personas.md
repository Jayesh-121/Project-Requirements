# Twitter Clone – User Personas

This document defines the primary user personas for the Twitter (X) clone application.

---

## Persona 1: Regular User (Content Creator)

**Name:** John  
**Age:** 21  
**Background:** College student  

### Goals
- Share thoughts and updates
- Follow people of interest
- Engage via likes, replies, and retweets
- Build a small audience

### Pain Points
- Wants simple and fast posting
- Needs clear and timely notifications
- Doesn’t want a complicated UI

### Key Features Used
- Signup / Login
- Create tweets
- Like, reply, and retweet tweets
- Follow / unfollow users
- View timeline
- Receive notifications

---

## Persona 2: Reader / Consumer

**Name:** Alice  
**Age:** 26
**Background:** Working professional  

### Goals
- Stay updated with news and trends
- Follow specific creators
- Browse content without posting much

### Pain Points
- Needs a clean and clutter-free feed
- Wants relevant content shown first

### Key Features Used
- Login
- Follow users
- View timeline
- Like tweets
- Search users and tweets

---

## Persona 3: Admin / Moderator

**Name:** Admin  
**Role:** Platform maintainer  

### Goals
- Maintain platform integrity
- Handle reports and abusive behavior
- Manage users and content

### Key Features Used
- View all users
- Suspend or ban users
- Remove tweets
- View reported content
